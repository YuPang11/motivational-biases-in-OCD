# Data and Code Availability

This repository contains the data and code used to reproduce the behavioral, EEG, computational modeling, and figure-generation analyses reported in the manuscript.

The analyses were conducted using MATLAB R2022b, EEGLAB 2022.0, FieldTrip v20211209, R 4.4.3, IBM SPSS Statistics 22.0, and MRIcroGL. Computational models were fitted using hierarchical Bayesian reinforcement-learning models.

## Software

The following software and packages were used:

- MATLAB R2022b
- EEGLAB 2022.0
- FieldTrip v20211209
- R 4.4.3
- R packages: `lme4`, `psych`, and `lmerTest`
- IBM SPSS Statistics 22.0
- MRIcroGL
- Stan/RStan or related Stan-based modeling scripts

## Figure generation

### Figure 1

Figure 1 illustrates the motivational Go/NoGo learning task structure, including cue presentation, response mapping, fixation, probabilistic feedback, and inter-trial interval.

The figure summarizes the four cue types generated by crossing valence and required action: Go-to-Win, Go-to-Avoid, NoGo-to-Win, and NoGo-to-Avoid. It also illustrates the response options and probabilistic feedback contingencies.

This figure is a schematic illustration of the experimental design and was generated from the task structure rather than from trial-level behavioral or EEG data.

### Figure 2

Figure 2 was generated from the behavioral data stored in `data/behavior`.

The behavioral data were first processed using `code/behavior/data.m`. This script extracts and organizes behavioral variables required for statistical analysis and figure generation, including trial-level cue type, response, accuracy, outcome, and condition labels.

The processed MATLAB data files were then used to generate Figure 2 using `code/behavior/behaviorplot.m`.

Figure 2 summarizes behavioral performance in the motivational Go/NoGo learning task.

### Figure 3

Figure 3 was generated from the behavioral data stored in `data/behavior`.

Computational modeling was conducted using scripts in `code/computational_modelling/modelrun`. These scripts fit hierarchical Bayesian reinforcement-learning models to trial-by-trial choices in the motivational Go/NoGo task. The models estimate parameters related to instrumental learning, Pavlovian bias, action bias, and choice stochasticity.

Figure 3 was generated using scripts in `code/computational_modelling/plot/M5_hcvsocd`.

Figure 3 presents behavioral computational modeling results, including model-derived parameters and group comparisons between healthy controls and patients with obsessive-compulsive disorder.

### Figure 4

Figure 4 was generated from the EEG data stored in `data/egi`.

EEG preprocessing was conducted using `code/EEGprocessing/eegpreprocess.m`. This script implements filtering, re-referencing, epoching, visual trial rejection, independent component analysis correction, bad-channel interpolation, and surface Laplacian transformation.

Time-frequency decomposition was then performed using `code/EEGprocessing/TF (subject-level).m` and `code/EEGprocessing/TF (group-level).m`. These scripts compute cue-locked and response-locked theta-band activity at the subject and group levels.

Figure 4A and Figure 4B were generated using `code/EEGprocessing/eegfigureplot.m`.

In addition, EEG variables used in computational modeling were prepared from the processed behavioral and EEG data. The script `code/behavior/data.m` generates Stan-compatible EEG modeling input files. Model fitting was performed using scripts in `code/computational_modelling/modelrun`, which generate fitted model outputs for each candidate model.

Model comparison plots were generated using scripts in `code/computational_modelling/plot/WAICplot`. Posterior estimates from the winning models and the association between theta activity and conflict-related effects were visualized using scripts in `code/computational_modelling/plot/βplot`.

### Figure 5

Figure 5 was generated from the EEG data stored in `data/egi`.

Theta-band intersite phase synchrony was computed using `code/ISPS/ISPS (subject-level).m` and `code/ISPS/ISPS (group-level).m`. These scripts calculate theta phase synchrony between the midfrontal seed cluster and two target clusters: the lateral prefrontal cluster and the motor cluster. ISPS values were baseline transformed and aggregated at the subject and group levels.

Figure 5A and Figure 5B were generated using `code/ISPS/ISPSplot.m`.

EEG-computational modeling input files were generated using `code/computational_modelling/eegRdatarun.R`. This script converts the processed EEG and behavioral variables into Stan-compatible R data files for EEG-informed computational model fitting.

The EEG-computational models were fitted using `code/computational_modelling/RunR.R`, which generates fitted model outputs for each candidate model.

Model comparison plots for Figure 5C and Figure 5D were generated using the following scripts:

- `code/computational_modelling/plot/WAICplot/DrawHC_WAIC_Set2.m`
- `code/computational_modelling/plot/WAICplot/DrawHC_WAIC_Set3.m`
- `code/computational_modelling/plot/WAICplot/rawOCD_WAIC_Set2right.m`
- `code/computational_modelling/plot/WAICplot/rawOCD_WAIC_Set3right.m`

Figure 5 therefore combines group-level theta phase-synchrony analyses with EEG-informed computational model comparison.

### Figure 6

Figure 6 was generated from the winning EEG-computational models reported in Figure 5C and Figure 5D.

The regulatory-orientation index, Φreg, was calculated using `code/Regulatory orientation/Φreg Calculation.R`. This script derives Φreg from the relevant model parameters estimated in the winning EEG-computational models.

Figure 6A and Figure 6B were generated using `code/Regulatory orientation/Φreg plot.m`.

Associations between Φreg and clinical symptom measures were tested using `code/Regulatory orientation/relation.m`.

Figure 6C was generated using MRIcroGL with the MNI152/ICBM152 standard brain template. The brain templates were generated in MRIcroGL from the MNI152/ICBM152 standard template.

## Reproducibility notes

To reproduce the analyses, users should run the scripts in the following general order:

1. Process behavioral data using `code/behavior/data.m`.
2. Generate behavioral plots using `code/behavior/behaviorplot.m`.
3. Fit behavioral computational models using scripts in `code/computational_modelling/modelrun`.
4. Generate behavioral model plots using scripts in `code/computational_modelling/plot/M5_hcvsocd`.
5. Preprocess EEG data using `code/EEGprocessing/eegpreprocess.m`.
6. Run time-frequency analyses using `code/EEGprocessing/TF (subject-level).m` and `code/EEGprocessing/TF (group-level).m`.
7. Generate theta-power figures using `code/EEGprocessing/eegfigureplot.m`.
8. Compute theta-band ISPS using `code/ISPS/ISPS (subject-level).m` and `code/ISPS/ISPS (group-level).m`.
9. Generate ISPS figures using `code/ISPS/ISPSplot.m`.
10. Prepare EEG-computational modeling input files using `code/computational_modelling/eegRdatarun.R`.
11. Fit EEG-computational models using `code/computational_modelling/RunR.R`.
12. Generate model comparison plots using scripts in `code/computational_modelling/plot/WAICplot`.
13. Generate posterior parameter plots using scripts in `code/computational_modelling/plot/βplot`.
14. Compute Φreg using `code/Regulatory orientation/Φreg Calculation.R`.
15. Generate Φreg plots using `code/Regulatory orientation/Φreg plot.m`.
16. Test associations between Φreg and clinical symptom measures using `code/Regulatory orientation/relation.m`.
17. Generate brain-template visualizations for Figure 6C in MRIcroGL.
